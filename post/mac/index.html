前言

大学时期折腾过黑苹果，VMware vSphere也搭建了虚拟环境测试，现在除了满大街的iPhone外，使用MacBook的朋友也越来越多，尽管对我而言也是触手可及，但总觉得少了当初的某种特殊追求。所以严格来说我没有资格来推荐Mac/iOS上的应用，希望使用苹果的你能帮助我一起补充。

扩展阅读中推荐了小闷的水果店（appled.cc）- 做最靠谱的苹果代购。

Let’s fix it together.

Mac OS X

推荐的不多，其它部分可参考扩展阅读

Office 2011 - http://pan.baidu.com/s/1kTqXqkv

  • iWork和Keynote也应该是大家常用的神器吧

VMware Fusion - http://www.0daydown.com/?s=VMware+Fusion

  • 不喜欢BootCamp那就直接用VMware虚拟机玩Windows吧
  • Parallels Desktop for Mac或许也是你的首选

Adobe Photoshop - http://pan.baidu.com/s/1kTEDMB1

  • Adobe Photoshop CS6 Extended for Mac，PS必备

百度输入法 For Mac - http://wuxian.baidu.com/input/mac.html

  • 腾讯因战略需要拱手想让输入法与搜狗，只剩百度与之匹敌了，百度也愈发强大

QQ For Mac - http://im.qq.com/macqq/

  • 静一些，近一点

迅雷 For Mac - http://mac.xunlei.com/

  • 纯粹的下载

QQ电脑管家 For Mac - http://mac.gj.qq.com/

  • 以简奴繁，和360又干上了

GoAgentX - http://goagentx.com

  • GoAgentX is a GUI application on OS X.
